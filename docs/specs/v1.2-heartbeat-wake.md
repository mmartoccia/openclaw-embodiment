# Spec: HeartbeatWake (v1.2)

**Location:** `openclaw_embodiment/core/trigger.py`
**Status:** Implemented

---

## Purpose

Eliminate up to 30 seconds of latency between a device event and the agent's first response. When `TriggerDetector` reaches CAPTURE state, it immediately calls `openclaw runtime heartbeat --now` via the gateway REST API, waking the agent before the context payload even arrives.

## Problem Solved

OpenClaw agents run on a heartbeat cycle (default ~30s interval). A trigger that fires at second 29 of the cycle would wait up to 30 more seconds for the agent to process it. `requestHeartbeatNow()` short-circuits this, reducing worst-case latency from 30s to under 1s.

---

## API Surface

```python
from openclaw_embodiment.core.trigger import TriggerConfig, TriggerDetector

# Config: heartbeat wake is automatic on CAPTURE state transition
config = TriggerConfig(
    polling_hz=25,
    # ... standard thresholds ...
)
detector = TriggerDetector(config)

# Manual wake (if needed):
result: bool = detector.request_heartbeat_now(reason="manual_test")
```

### `TriggerDetector.request_heartbeat_now()`

```python
def request_heartbeat_now(self, reason: str = "device_trigger") -> bool:
    """Wake the OpenClaw agent via gateway REST API.

    Enforces 5-second cooldown between calls.
    Returns True if wake dispatched, False if on cooldown or unavailable.
    """
```

### Gateway endpoint:
```
POST http://localhost:18799/heartbeat/trigger
Content-Type: application/json
{"reason": "<str>", "source": "embodiment_sdk"}
```

---

## Integration Points

- **TriggerDetector.update():** Calls `request_heartbeat_now("capture_trigger")` on every CAPTURE transition.
- **OpenClaw gateway:** Must be running on `localhost:18799` (standard OpenClaw setup).
- **5-second cooldown:** `HEARTBEAT_COOLDOWN_S = 5.0` -- prevents spam from high-rate sensors.

---

## Failure Modes

| Failure | Handling |
|---------|----------|
| Gateway not running | Caught; returns False; pipeline continues |
| Network timeout | 2s timeout; caught; returns False |
| Cooldown active | Returns False immediately (no network call) |
| Gateway 4xx/5xx | Returns False; logged at DEBUG |
