# Spec: Reachy 2 Device Profile (v1.2)

**Location:** `openclaw_embodiment/hal/reachy2_reference.py` + `profiles/reachy2.yaml`
**Status:** Implemented (hardware validation pending)

---

## Purpose

Full humanoid HAL for the Pollen Robotics Reachy 2 platform. Same HAL ABCs as Reachy Mini, richer actuator map: 7-DOF arms × 2, grippers, 3-DOF neck, stereo cameras, and optional wheeled mobile base.

---

## Hardware

- **Arms:** 7-DOF per arm (shoulder pitch/roll, elbow yaw/pitch, wrist roll/pitch/yaw + gripper)
- **Neck:** 3-DOF (pan, tilt, roll) with real encoder feedback
- **Cameras:** Stereo (left + right, depth-capable, 640x480 default)
- **Microphone:** Array with Direction of Arrival (DoA) support
- **Speaker:** TTS/audio playback
- **Face display:** LED antenna expressions + screen
- **Mobile base:** Optional wheeled base
- **Transport:** HTTP/gRPC via reachy2-sdk

### Extended `ActuatorHal` capabilities (Reachy 2)

```python
actuator.execute(ActuatorCommand(action="move_arm", params={
    "side": "right",
    "joints": {"shoulder_pitch": 15.0, "elbow_pitch": -30.0},
    "speed": 0.5,
}))

actuator.execute(ActuatorCommand(action="open_gripper", params={"side": "left"}))
actuator.execute(ActuatorCommand(action="close_gripper", params={"side": "right"}))

actuator.execute(ActuatorCommand(action="move_head", params={
    "neck_pan": 20.0, "neck_tilt": -10.0, "neck_roll": 0.0,
}))

actuator.execute(ActuatorCommand(action="set_mobile_base_velocity", params={
    "vx": 0.1, "vy": 0.0, "omega": 0.05,
}))
```

### Supported actions
| Action | Description |
|--------|-------------|
| `move_head` | 3-DOF neck pan/tilt/roll |
| `move_arm` | 7-DOF arm (side: left/right) |
| `open_gripper` | Open gripper (side: left/right) |
| `close_gripper` | Close gripper (side: left/right) |
| `animate_antennas` | LED antenna animations |
| `set_expression` | Face LED expressions |
| `set_mobile_base_velocity` | vx, vy, omega |
| `stop_all` | Emergency stop |

### Joint constants
```python
REACHY2_HEAD_JOINTS = ["neck_pan", "neck_tilt", "neck_roll"]
```

## Data Schemas

Uses standard HAL ABCs from `hal/base.py`. No new schemas.

## Integration Points

- **TransportHal:** HTTP (`get_expected_latency_ms() = 10ms`)
- **MicrophoneHal:** DoA-capable (`get_doa()` returns azimuth/elevation)
- **STT bridge:** `transcribe()` via `OpenClawSTTBridge`
- **TriggerArbiter:** DoA from mic triggers head orient before visual capture

---

## Failure Modes

| Failure | Handling |
|---------|----------|
| reachy2-sdk not installed | `ImportError` at HAL instantiation |
| Robot offline | `ConnectionError` propagated |
| Arm joint out of range | `ValueError` from reachy2-sdk |
| Mobile base not installed | `ActuatorResult(success=False, error="no mobile base")` |
