# Spec: Even G2 Audio Path Completion (v1.2)

**Location:** `openclaw_embodiment/hal/even_g2_reference.py`
**Status:** Implemented (hardware validation pending)

---

## Purpose

Complete the audio capture and playback path for Even Realities G2 smart glasses. The G2 uses LC3 audio codec over BLE for microphone capture and A2DP for bone conduction speaker playback.

## Problem Solved

The G2 profile had BLE display and transport implemented, but the audio path (mic -> STT, TTS -> bone conduction) was incomplete. Without this, G2 was a display-only device. Completing audio makes G2 a full bidirectional voice interface.

---

## API Surface

### `G2MicrophoneHAL`
```python
class G2MicrophoneHAL(MicrophoneHal):
    def initialize(self, sample_rate: int = 16000, channels: int = 1) -> None: ...
    def start_recording(self) -> None: ...
    def stop_recording(self) -> None: ...
    def get_buffer(self, duration_ms: int) -> AudioChunk: ...
    def transcribe(self, audio: AudioChunk, language: str = "en") -> str: ...
    def transcribe_stream(self, stream) -> Iterator[str]: ...
    def shutdown(self) -> None: ...
```

BLE Audio characteristic: `0x0000F1-0000-1000-8000-00805f9b34fb`
LC3 frame size: 40 bytes (10ms @ 16kHz, mono)
Decoding: python-lc3 library (if available), fallback raw bytes

### `G2AudioOutputHAL` (planned)
```python
class G2AudioOutputHAL(AudioOutputHal):
    def initialize(self, sample_rate: int = 22050, channels: int = 1) -> None: ...
    def play(self, audio_data: bytes, format: str = "PCM_S16LE", ...) -> None: ...
    def stop(self) -> None: ...
    def is_playing(self) -> bool: ...
    def speak_agent_response(self, response: AgentResponse) -> None: ...
    def shutdown(self) -> None: ...
```

Playback via A2DP (standard Bluetooth audio routing via OS audio stack).
TTS audio generated on host, routed to paired G2 via A2DP profile.

## Data Schemas

Uses standard `AudioChunk` from `hal/base.py`.

LC3 audio format within AudioChunk:
```
AudioChunk(
    sample_rate=16000,
    channels=1,
    format="LC3",
    data=<raw LC3 frames>,
)
```

---

## Protocol Details

**Microphone:**
- Characteristic: `0x0000F1-0000-1000-8000-00805f9b34fb`
- Subscribe NOTIFY to receive LC3 audio frames
- Each frame: 40 bytes, 10ms audio @ 16kHz mono
- Notify rate: 100 notifications/second

**Speaker (A2DP):**
- Standard OS Bluetooth audio routing
- G2 appears as Bluetooth audio device
- Route TTS output to G2 via `pyaudio` or OS audio API
- No custom BLE characteristic needed

## Integration Points

- **STT bridge:** `transcribe()` delegates to `OpenClawSTTBridge`
- **G2TransportHAL:** Already implemented; audio path is independent
- **TriggerArbiter:** Audio energy from `G2MicrophoneHAL` feeds `AudioTriggerDetector`
- **AgentResponseListener:** TEXT responses routed to `G2AudioOutputHAL.speak_agent_response()`

---

## Failure Modes

| Failure | Handling |
|---------|----------|
| BLE disconnect | `G2MicrophoneHAL` raises; pipeline catches |
| LC3 decoder not installed | Raw bytes returned; transcription degrades |
| A2DP not paired | `G2AudioOutputHAL.play()` fails silently |
| No speech detected | `transcribe()` returns `""` |
